# 需求文档

## 项目一句话描述

一个 Mac 上的浮动便签 App：始终可以置顶悬浮在其他网站/桌面/App 之上，方便随时记笔记（例如看法语学习视频时记笔记、浏览网页时摘抄认同的文字），所有笔记自动保存，按时间倒序排列。

## 使用者

仅供本人（用户）在自己的 Mac 上使用，不需要跨平台、不需要分享给他人、不需要账号登录、不需要云同步。

## 核心需求清单

1. **悬浮便签框**：一个小窗口，始终置顶显示在最上层，包括：
   - 浏览网页时
   - 使用其他 App 时
   - **全屏看视频时**（重点场景，例如全屏法语学习视频）
2. **显示/隐藏**：悬浮框默认一直显示；同时支持全局快捷键（不需要先切到这个 App）快速隐藏/显示，默认快捷键 **⌃⌥N（Control+Option+N）**。
3. **笔记列表主窗口**：另一个窗口，展示所有保存过的笔记，**按时间倒序排列**（最新的在最上面）。
4. **内容类型**：
   - 第一阶段：纯文字（手动输入，或从网页/视频字幕复制粘贴）
   - 之后阶段：支持粘贴/拖入图片或截图
5. **分类/标签**：笔记可以打简单标签（例如"法语学习" vs "常规"），列表可以按标签筛选查看。
6. **数据存储**：完全存在本地 Mac 上，不需要登录账号，不需要 iCloud 同步。
7. **视觉风格**：简洁、直观；**主色调为粉色**；操作要一目了然（保存、打开列表、选标签都要一眼看懂怎么用），不用图标猜谜。

## 明确不做的事情（Out of Scope）

- 不做跨平台（Windows/网页版）
- 不做分享给他人使用 / 不需要上架 App Store / 不需要付费开发者账号公证
- 不做账号系统、不做云同步
- 第一阶段不做图片/截图支持（列入后续阶段）
- 不做复杂的多级文件夹分类，只做简单标签
